# 🥖 Aliança Pães e Doces – Landing Page

Landing page moderna desenvolvida para uma padaria local, com foco em aumentar conversões, facilitar pedidos via WhatsApp e melhorar a presença digital do negócio.

---

## 🚀 Objetivo

O projeto foi criado com o objetivo de resolver um problema comum em pequenos negócios locais: a falta de uma presença digital eficiente.

A página foi pensada para:

* Atrair clientes logo na primeira impressão
* Facilitar o acesso ao cardápio
* Direcionar pedidos diretamente para o WhatsApp
* Aumentar a taxa de conversão de visitantes em clientes

---

## 🧠 Estratégia

Ao invés de apenas apresentar informações, o site foi estruturado como uma **página de vendas**, com foco em ação rápida do usuário.

Principais decisões:

* Design simples e direto
* Botões de ação (CTA) bem visíveis
* Conteúdo objetivo
* Navegação intuitiva

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3 (Flexbox & Grid)
* JavaScript (Vanilla)

---

## 📱 Funcionalidades

* Layout responsivo (mobile-first)
* Botão de pedido direto no WhatsApp
* Seção de cardápio com destaque de produtos
* Integração com redes sociais
* Scroll suave entre seções
* Animações leves para melhor experiência
